# Wiki

База знаний для вашего проекта. Модуль позволяет создавать справочные статьи и организовывать их по категориям. Каждая статья поддерживает Markdown-разметку, автоматическое оглавление по заголовкам, теги для удобного поиска и систему оценки полезности материала.

На главной странице базы знаний отображаются все категории со списком статей внутри. Встроенный поиск работает по заголовкам, описаниям и содержимому статей. При просмотре статьи сбоку выводится навигация по разделам и информация об авторе и дате последнего обновления.

В админ-панели можно управлять категориями и статьями, задавать иконки для категорий, настраивать порядок сортировки и отслеживать статистику просмотров.

---

Knowledge base for your project. The module allows you to create reference articles and organize them into categories. Each article supports Markdown formatting, automatic table of contents based on headings, tags for convenient search, and a helpfulness rating system.

The main page displays all categories with their articles listed inside. Built-in search works across titles, descriptions, and article content. When viewing an article, a sidebar shows section navigation along with author information and last update date.

The admin panel provides tools for managing categories and articles, setting category icons, configuring sort order, and tracking view statistics.

## Installation

Download the latest release and install it via the Flute CMS admin panel.

Current version: **1.0.3**
